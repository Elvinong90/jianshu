## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:4200
ng serve

# build for production
ng build --prod

```

For detailed explanation on how things work, checkout the [docs for angular-cli](https://github.com/angular/angular-cli).
